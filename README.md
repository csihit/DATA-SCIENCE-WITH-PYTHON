# DATA-SCIENCE-WITH-PYTHON

Welcome to the **Data Science with Python Workshop** 
This repository contains all the **notebooks (.ipynb files)** used during the workshop.

---

## Topics Covered

- Introduction to Data Science
- Python Basics (Variables, Data types, Operators, String, Slicing, Condition Statements, List & Tuples, Dictionaries & Set, Loops, Functions)
- NumPy  
- Pandas
- Data Visualization (Matplotlib)

---

## Files Included

- `Session1.ipynb` → Python Basics 
- `Numpy1.ipynb` → Basics of NumPy
- `Numpy2.ipynb` → Advanced NumPy  
- `Numpy_and_Pandas.ipynb` → Combined concepts  
- `Numpy_and_pandas_2.ipynb` → Practice notebook  
- `Heart_Disease_Prediction.csv` → Dataset  

---

## How to Download This Repository 

1. Click on the Code button (green colour).
2. Click on the Download ZIP.
3. Extract the ZIP file on your system.

---

## No Installation Needed!

You can run everything directly on **Google Colab**.

---

## How to Use (Google Colab)

### Step 1 : Open Google Colab

Go to : https://colab.research.google.com  

---

### Step 2 : Upload Notebook

1. Click on **Upload** tab  
2. Upload the `.ipynb` file from your system  
3. The notebook will open automatically.  

---

### Step 3 : Run the Code

- Click ▶️ Run button on each cell  
- OR press **Shift + Enter**

---

## How to Use Dataset in Colab

- Click 📁 (left side panel)
- Click Upload icon ⬆️
- Select file 'Heart_Disease_Prediction.csv'

- NOTE :- Files will be deleted after runtime ends. So students must upload again each time
